# Yuzuki MD v2

A WhatsApp bot built on the Baileys/Focashi library with features including media downloaders, AI commands, group tools, Pinterest search, and more.

## Stack
- Node.js 20+ (ESM modules)
- Focashi (socketon) — WhatsApp library forked from Baileys
- Supabase — database
- Entry point: `src/index.js`
- Run: `npm start`

## Workflow
- **Start application** — runs `npm start`, console output type

## Notes
- `socketon` is installed from GitHub (`AhmadAkbarID/focashi`) — needs chalk v4 copied into `node_modules/socketon/node_modules/chalk/`
- Git remote `origin` points to `https://github.com/KyokaAizen665/Yuzuki-Md-V2`
